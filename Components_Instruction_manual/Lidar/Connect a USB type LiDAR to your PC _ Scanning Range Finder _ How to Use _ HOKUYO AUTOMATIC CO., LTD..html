How to UseScanning Range Finder

Connect a USB type LiDAR to your PC

[Preparation] Download URG USB Driver

A dedicated USB driver is required to connect a USB type LiDAR to a Windows PC. You can download the driver from the "Download" item on each product page. The area setting type is enclosed with the area setting application. If you are using MacOS or Linux, please refer to the communication specifications of each product.

Install the URG USB Driver

After downloading the URG USB Driver and unzipping the zip file, you will find "README.txt" and "URG_USB_Driver.inf" in the folder. Even if you double-click "URG_USB_Driver.inf", right-click → open, etc., it will not be installed. The installation method will be explained from the next section. * If you are using Windows 8 or 8.1, you need to set the signature in advance. If you are using Windows 10, the driver will be handled differently. For details, please refer to the notes on connecting a USB type scanner to a Winsows 8, 8.1, 10 PC.

[Procedure 1] Device Manager

Turn on the sensor, connect it to your PC with a USB cable, and open "Device Manager". If the driver is not installed, "URG-Series USB Driver" will be displayed on the "Other Devices" tab. Right-click "URG-Series USB Driver" → click Properties.

[Procedure 2] Properties of URG-Series USB Driver

Click "Update Driver" on the property screen of URG-Series USB Driver.

[Procedure 3] Driver update

On the driver software update screen, click "Browse my computer for driver software". Click Browse on the next screen, select the folder containing "URG_USB_Driver.inf" and click "Next".

[Procedure 4] Windows Security

The Windows security screen will appear. Click "Install this driver software". This completes the settings.
